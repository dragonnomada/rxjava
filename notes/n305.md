# 305 - Operadores Booleanos

Por [Alan Badillo Salas](https://www.nomadacode.com)

## `all()`

```java
Observable.range(1, 5)
    .all(i -> i % 2 == 0)
    .subscribe(System.out::println)
```

## `any()`

```java
Observable.range(1, 5)
    .any(i -> i % 2 == 0)
    .subscribe(System.out::println)
```

## `isEmpty()`

```java
Observable.range(1, 0)
    .isEmpty()
    .subscribe(System.out::println)
```

## `contains()`

```java
Observable.range(1, 5)
    .contains(3)
    .subscribe(System.out::println)
```

## `sequenceEqual()`

```java
Observable.sequenceEqual(
    Observable.range(1, 5),
    Observable.just(1, 2, 3, 4, 5)
).subscribe(System.out::println)
```
