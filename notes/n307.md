# 307 - Operadores para el Manejo de Errores

Por [Alan Badillo Salas](https://www.nomadacode.com)

## `onErrorReturn()`

```java
Observable.range(1, 100)
    .takeWhile(i -> i <= 5)
    .subscribe(System.out::println)
```

## `onErrorReturnItem()`

```java
Observable.range(1, 100)
    .takeWhile(i -> i <= 5)
    .subscribe(System.out::println)
```

## `onErrorResumeWith()`

```java

    .subscribe(System.out::println)
```

## `onErrorResumeNext()`


